# 是模玩店！积分会员小程序 — 使用文档

> 适用对象：店主（管理员）、店员、顾客。
> 小程序定位：消费攒积分 → 积分兑换好礼/优惠券 → 到店核销。纯积分、零支付。

---

## 一、角色说明
| 角色 | 能做什么 | 怎么获得身份 |
|---|---|---|
| **顾客** | 攒积分、签到、浏览、兑换、看券、会员码 | 微信进入小程序即自动成为会员 |
| **店员** | 给会员加/扣积分、核销兑换券 | 管理员在「店铺管理-店员」里添加 |
| **管理员（店主）** | 店员全部功能 + 上架商品/轮播/活动、管理店员 | 用动态口令在「员工入口」认证 |

---

## 二、顾客使用指南

### 1. 进入与入会
- 微信里打开小程序即自动注册会员，**新客入会送 120 积分**。
- 首次进入会在「积分明细」看到一条「入会 +120」。

### 2. 每日签到
- 首页点「签到」→ 每天可领一次积分（默认 +10）。
- 当天重复点会提示「今天已签到」。

### 3. 浏览与搜索
- 首页：品牌头图、商品橱窗（点商品进详情）。
- 点顶部搜索栏 → 进搜索页：可按**关键词**搜，按**类别**（优惠券/周边/贴纸/盲盒）筛，按**积分高低**排序。

### 4. 兑换商品
- 进商品详情 → 点「立即兑换」→ 确认 → 生成**兑换码**。
- 积分不足 / 已兑完 / 已下架时按钮自动置灰并提示。

### 5. 查看与核销兑换券
- 「我的 → 我的兑换」查看所有兑换单（待核销/已完成/已取消）。
- 待核销的可「查看券码」，**到店向店员出示券码**核销。

### 6. 会员码 / 个人资料
- 底部「会员码」：出示给店员用于加积分。
- 「我的」页点头像/昵称可修改（头像存云端，换设备不丢）。

---

## 三、店员使用指南

### 1. 成为店员
- 由管理员在「店铺管理-店员」里输入你的**会员号**添加（会员号在你自己的「会员码」页查看复制）。
- 添加后重新进入小程序，「我的」页底部出现「店员工作台」。

### 2. 给会员加 / 扣积分
- 进「店员工作台 → 查询会员」：输入或扫会员号 → 查出会员。
- 输入分值 → 点「加积分 / 扣积分」。扣分不能扣成负。

### 3. 核销兑换券
- 「店员工作台 → 核销兑换券」：输入或扫顾客的兑换码 → 核销。
- 已核销/不存在的码会提示，杜绝重复核销。

## 四、管理员（店主）使用指南

### 1. 成为管理员（动态口令登录）
1. 在手机装一个**身份验证器 App**（Google / 腾讯 / 微软 Authenticator）。
2. 手动添加（基于时间），密钥由开发者提供（设置在 claimAdmin 云函数）。
3. 在「我的」页底部**连点版本号 7 下** → 进员工入口 → 输入验证器上的 **6 位动态码** → 成为管理员。
> 动态码每 30 秒变一次，安全；连错 5 次锁 5 分钟。

### 2. 店铺管理（手机上全搞定）
「我的 → 店铺管理」，四个标签页：
- **商品**：新增/编辑/删除商品（名称、分类、所需积分、库存、描述、上架开关、商品图）。下架的商品顾客端不显示。
- **轮播**：管理首页轮播图（标题、排序、图）。
- **动态**：发布新品到货 / 活动预告 / 店铺公告。
- **店员**：输入会员号把人设为店员或管理员；可删除（不能删自己）。

### 3. 退出 / 切换
- 「我的 → 退出员工身份」可退出，便于交接或测试。

## 五、初次部署 / 上线准备（开发者操作）
1. 微信开发者工具打开项目，开通云开发，填环境 ID。
2. 部署全部云函数，运行 `init` 建表灌示例数据。
3. 控制台配置各集合安全规则（见 SECURITY.md）。
4. 用动态码成为首个管理员。
5. 上架真实商品、配置轮播与活动。
6. 上传体验版 → 内部验收 → 提交审核 → 发布。

## 六、常见问题
- **Q：顾客需要下载 App 吗？** 不需要，微信里直接用。
- **Q：要联网服务器吗？** 不要，用微信云开发，免服务器免备案。
- **Q：积分会丢吗？** 全部存云端，换手机/重装不丢。
- **Q：会被刷积分吗？** 已做三轮安全加固，所有积分变动服务端校验，防并发刷单、防越权。
- **Q：店员能乱加积分吗？** 店员操作有记录可审计，单次加分有上限；店员由管理员授权，可随时移除。
