# 是模玩店！积分会员小程序 — 项目阶段说明

## 一句话
为模玩店打造的**积分会员小程序**：顾客消费攒积分 → 兑换好礼/优惠券 → 到店核销；店主在手机上即可管理。纯积分、零支付，**免服务器、免备案**，运维成本极低。

## 现在已经能跑通的（可真机体验）
**顾客端**
- 进入即注册会员，**新客入会送 120 积分**
- **每日签到**领积分
- 首页：品牌头图 + 搜索 + 商品橱窗
- **搜索 + 筛选**：按关键词、类别、积分高低筛选商品
- **积分兑换**：选商品 → 出兑换码 → 到店核销
- 我的积分 / 积分明细 / 我的兑换 / 会员码 / 个人资料

**店主/店员端**
- **动态口令登录**（手机验证器，安全）
- **店铺管理**：商品 / 轮播图 / 活动动态 / 店员，全在手机上增删改、传图
- **店员工作台**：扫会员码加积分、核销兑换券

**底层质量**
- 暗黑科技风视觉，统一品牌
- **三轮安全审计**：防刷积分、防超兑、防越权、防暴力破解，全部服务端校验
- 完整的进度文档与测试清单

## 完成上线还需要的（经费覆盖的部分）
1. **真实素材**：商品实拍图、Banner、部分图标精修
2. **真实数据录入**：店内真实商品 / 积分价 / 活动
3. **合规与提审**：小程序认证、隐私协议配置、提交微信审核、发布
4. **小程序认证费**：微信平台 ¥300/年（企业/个体）
5. （可选增强）会员码真二维码扫码、连续签到奖励、数据看板等

## 体验方式
- 扫描随附的**体验版二维码**即可在手机上试用（已连真实云端数据）
- 管理员功能由开发者现场演示
